# Chestnut Editor v0.3.0

---

## English

A desktop editor for local Markdown and Excalidraw files — round-trip Markdown bundles, sharper file-tree workflow, and smoother daily editing.

Chestnut Editor reads and writes plain files in a folder you choose. No proprietary database, no cloud required.

### Download

| Item | Details |
|------|---------|
| **Installer** | `Chestnut_0.3.0_x64-setup.exe` (Windows x64, NSIS) |
| **Requirements** | 64-bit Windows 10+; WebView2 (downloaded during install if missing) |
| **Default vault** | `~/.chestnut` on first launch |

**Build output:**

`apps/desktop/src-tauri/target/x86_64-pc-windows-msvc/release/bundle/nsis/Chestnut_0.3.0_x64-setup.exe`

### What's New in v0.3.0

#### Markdown export & import

- **Export as Markdown** from the file tree: writes `target/{NoteName}/{NoteName}.md` plus co-located images
- Image refs in exported notes use same-directory paths (`image.png`) for use in other tools
- Export progress dialog (same style as PDF export); after export, the file tree selects the new note under `target/`
- **Import Markdown folder** from the top toolbar (right of Settings): pick a folder with `.md` + images in one directory and convert to vault **note + `{NoteName}_pic/`** automatically

#### File tree & navigation

- **Folder selection** is separate from the open file; clicking a folder no longer clears the editor’s active note highlight incorrectly
- New notes, drawings, and folders are created in the **selected folder** (or next to the active file when nothing is selected)
- After create, the tree **reveals and selects** the new item and enters **rename** immediately
- Selection highlight applies **on click** (no delay)
- **Reveal in file manager** works reliably for vault files on Windows

#### Excalidraw & UI polish

- Excalidraw UI **follows the app language** (English / 简体中文)
- Context menus and sidebar tooltips no longer clip at viewport edges
- Fix stray **Quick Open / Search** shortcuts firing when refocusing the app after double-tap Shift

### Upgrading from v0.2.0

- No vault migration required; existing notes and `_pic` folders are unchanged
- Exported bundles live under the vault `target/` folder (same as PDF export)
- Import expects a flat folder: one primary `.md` (prefer `{FolderName}.md`) and image files beside it

### Highlights from v0.2.0 (still included)

- Chestnut Editor branding, bilingual UI, light/dark theme, handwriting fonts
- Resizable file sidebar, drag-and-drop move, reveal in tree, Recycle Bin delete
- PDF export with preview; image viewer, lightbox, caption editing, optional `_pic` cleanup
- Editor context menu; quick open, search, shortcuts; Excalidraw in-app; plugins (experimental)

## 简体中文

面向本地 Markdown 与 Excalidraw 的桌面编辑器 —— Markdown 包导入导出、文件树交互更精细，日常编辑更顺手。

Chestnut Editor 在你选择的文件夹中直接读写普通文件，无需专有数据库，也不依赖云端。

| **系统要求** | 64 位 Windows 10 及以上；缺少 WebView2 时安装程序会自动下载 |
| **默认知识库** | 首次启动打开 `~/.chestnut` |

**构建产物：**

`apps/desktop/src-tauri/target/x86_64-pc-windows-msvc/release/bundle/nsis/Chestnut_0.3.0_x64-setup.exe`

### v0.3.0 新内容

#### Markdown 导入与导出

- 文件树右键 **导出为 Markdown**：生成 `target/{笔记名}/{笔记名}.md`，图片与 md **同目录**存放
- 导出 md 中图片引用为同目录相对路径（`image.png`），便于在其他编辑器中使用
- 导出过程显示与 PDF 相同的**进度对话框**；完成后在文件树中**选中** `target/` 下对应 md
- 顶栏 **设置图标右侧** 提供 **导入**：选择 md 与图片同目录的文件夹，自动转为知识库 **笔记 + `{笔记名}_pic/`** 结构

#### 文件树与导航

- **文件夹选中**与当前打开文件分离；点选文件夹时编辑器仍显示原笔记
- 新建笔记、绘图、文件夹会创建在**当前选中文件夹**（未选中时则在当前文件所在目录）
- 新建后文件树**定位并选中**新项，并**立即进入重命名**
- 点击选中**即时高亮**（无延迟）
- Windows 下 **在文件管理器中打开** 对 vault 内文件更可靠

#### Excalidraw 与体验优化

- Excalidraw 界面语言**跟随应用语言设置**（中/英）
- 右键菜单与侧栏 tooltip 不再在视口边缘被裁切
- 修复切回应用时误触发**快速打开 / 搜索**（双击 Shift 残留）的问题

### 从 v0.2.0 升级说明

- 无需迁移知识库；现有笔记与 `_pic` 目录保持不变
- 导出结果位于知识库 `target/` 目录（与 PDF 导出相同）
- 导入要求**扁平文件夹**：一个主 `.md`（优先 `{文件夹名}.md`）及同目录图片文件

### v0.2.0 已有能力（本版保留）

- Chestnut Editor 品牌、中英界面、浅/深色主题、手写体字体
- 可调整宽度的文件侧栏、拖拽移动、树内定位、回收站删除
- PDF 导出与预览；图片查看、灯箱、说明编辑、可选 `_pic` 清理
- 编辑器右键菜单；快速打开、搜索、快捷键；应用内 Excalidraw；插件（实验性）
