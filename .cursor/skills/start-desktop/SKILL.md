# 以桌面应用启动 Boke

与 [start-boke](../start-boke/SKILL.md) 相同：在仓库根目录执行 `pnpm dev`（Tauri dev）。

打包：`pnpm build:desktop` → `apps/desktop/src-tauri/target/release/boke.exe`
