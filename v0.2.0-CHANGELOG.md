# Chestnut Editor v0.2.0

---

## English

A desktop editor for local Markdown and Excalidraw files — clearer identity, richer file workflow, and a warmer writing experience.

Chestnut Editor reads and writes plain files in a folder you choose. No proprietary database, no cloud required.

### Download

| Item | Details |
|------|---------|
| **Installer** | `Chestnut_0.2.0_x64-setup.exe` (Windows x64, NSIS) |
| **Requirements** | 64-bit Windows 10+; WebView2 (downloaded during install if missing) |
| **Default vault** | `~/.chestnut` on first launch |

**Build output:**

`apps/desktop/src-tauri/target/x86_64-pc-windows-msvc/release/bundle/nsis/Chestnut_0.2.0_x64-setup.exe`

### What's New in v0.2.0

#### Brand & identity

- Project renamed from **Boke** to **Chestnut Editor**
- New hand-drawn chestnut icon for window, taskbar, and installer
- Bilingual welcome READMEs in the vault (`README_en.md` / `README_cn.md`)

#### File tree & navigation

- Resizable, collapsible file sidebar (drag edge; double-click tab bar to toggle)
- Drag-and-drop to move files and folders, with ghost preview; drop outside tree moves to vault root
- Reveal active file in the tree; collapse all folders
- Delete moves items to the **Recycle Bin** (recoverable)
- Open files or images in the system file manager

#### Markdown & images

- Export notes to **PDF** with in-app preview and progress dialog
- Image viewer for file tree and in-note clicks
- Lightbox zoom; edit image captions in live mode
- Optional cleanup of orphaned files in `{NoteName}_pic/` folders
- Editor **context menu**: copy, paste, insert table / code / math / task list (native clipboard on desktop)

#### Appearance & settings

- **English** and **简体中文** UI
- Light / **dark theme** for Markdown and Excalidraw (preference saved)
- Font picker: Microsoft YaHei, Xiaolai, Yozai (handwriting fonts embedded)

#### Polish

- Redesigned toolbar with icon actions, editable vault path, copy path
- Tab bar scrolling and bulk close (others / left / right / all)
- Improved note layout, folder UX, and nested-path fixes

### Upgrading from v0.1.0 (Boke)

- App name is now **Chestnut Editor**; installer and exe use the **Chestnut** product name
- New default vault path: `~/.chestnut` (existing `~/.boke` vaults still work)
- Hidden config folder: `.chestnut/` (legacy `.boke/` is still recognized)
- Settings migrate automatically from the old localStorage key

### Highlights from v0.1.0 (still included)

- Local-first vault on disk; auto-save for Markdown and Excalidraw
- Live preview + source mode; wikilinks, embeds, tags, frontmatter
- Quick open, full-text search, configurable shortcuts
- Excalidraw editing in-app; paste images to `{NoteName}_pic/`
- Plugin support (experimental) via `.chestnut/plugins/`

面向本地 Markdown 与 Excalidraw 的桌面编辑器 —— 品牌更清晰，文件操作更顺手，写作体验更完整。

Chestnut Editor 在你选择的文件夹中直接读写普通文件，无需专有数据库，也不依赖云端。

### v0.2.0 新内容

#### 品牌与识别

- 项目由 **Boke** 正式更名为 **Chestnut Editor（栗子编辑器）**
- 全新手绘板栗图标（窗口、任务栏、安装包）
- 知识库内自动生成中英双语欢迎文档（`README_en.md` / `README_cn.md`）

#### 文件树与导航

- 文件侧栏可**拖拽调整宽度**、可**折叠**（双击标签栏可切换显示）
- **拖拽移动**文件与文件夹，带幽灵预览；拖到树外视为移到知识库根目录
- 在树中**定位当前文件**；**一键折叠**全部文件夹
- 删除移入**系统回收站**（可恢复）
- 在系统文件管理器中打开文件或图片

#### Markdown 与图片

- 笔记**导出 PDF**，支持应用内预览与进度对话框
- 文件树与笔记内点击图片可**查看大图**
- **灯箱放大**预览；实时模式下编辑**图片说明**
- 可选清理 `{笔记名}_pic/` 中未被引用的孤儿图片
- 编辑器**右键菜单**：复制、粘贴、插入表格 / 代码 / 公式 / 任务列表（桌面端使用原生剪贴板）

#### 外观与设置

- 界面支持 **English** 与 **简体中文**
- Markdown 与 Excalidraw **浅色 / 深色主题**，偏好自动保存
- 字体选择：微软雅黑、小赖体、悠哉体（内置手写体）

#### 体验优化

- 顶栏工具栏重设计：图标操作、可编辑知识库路径、复制路径
- 标签栏支持滚动与批量关闭（其他 / 左侧 / 右侧 / 全部）
- 笔记排版、文件夹交互与嵌套路径等多项修复与打磨

### 从 v0.1.0（Boke）升级说明

- 应用名称现为 **Chestnut Editor**；安装包与 exe 产品名为 **Chestnut**
- 新默认知识库路径为 `~/.chestnut`（已有 `~/.boke` 知识库可继续使用）
- 配置目录为 `.chestnut/`（仍兼容旧版 `.boke/`）
- 应用设置会自动从旧版 localStorage 迁移

### v0.1.0 已有能力（本版保留）

- 本地优先，磁盘即知识库；Markdown / Excalidraw 自动保存
- 实时预览 + 源码模式；双链、嵌入、标签、YAML frontmatter
- 快速打开、全文搜索、可自定义快捷键
- 应用内编辑 Excalidraw；粘贴图片保存至 `{笔记名}_pic/`
- 插件支持（实验性），位于 `.chestnut/plugins/`
